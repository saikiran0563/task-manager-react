# Task Manager React App

A simple and modern task management application built using React.js.

---

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Data saved using localStorage
- Responsive UI

---

## Tech Stack

- React.js
- JavaScript
- CSS
- React Hooks (useState, useEffect)
